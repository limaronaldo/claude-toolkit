Multi-Agent Context Output

Generate context files for AI coding tools beyond Claude Code.

Supported Agents

AgentOutput File
claudeCLAUDE.md (default)
cursor.cursor/rules/project.mdc
copilot.github/copilot-instructions.md
windsurf.windsurfrules
aider.aider/conventions.md
codexAGENTS.md

Usage

# Generate for a single agent
claude-primer --agent cursor --yes

# Generate for multiple agents
claude-primer --agent cursor,copilot,codex --yes

# Generate for all supported agents
claude-primer --agent all --yes

Output Formats

Agent files default to markdown. You can also output structured data:

# JSON format
claude-primer --agent copilot --format json --yes

# YAML format
claude-primer --agent copilot --format yaml --yes

Structured formats create a .claude-primer-context.{json,yaml} file with the full project analysis.

Agent File Content

Each agent file contains the same core project context adapted to the tool's conventions:

  • Project overview and tech stack
  • Directory structure
  • Common commands
  • Coding standards and conventions
  • Framework-specific patterns
