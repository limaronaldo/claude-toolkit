CLI Reference

Synopsis

claude-primer [target] [options]

Positional Arguments

ArgumentDescription
targetTarget directory (default: current directory)

Core Options

FlagDescription
--dry-runPreview changes without writing files
--forceOverwrite existing files (skip unchanged)
--force-allOverwrite all files unconditionally
--yes, -yAccept defaults, no interactive prompts
--plan-jsonOutput project analysis as JSON without writing
--reconfigureRe-run wizard (ignores saved .claude-setup.rc)

Content Options

FlagDescription
--with-readmeInclude README.md in generated files
--with-ralphGenerate Ralph integration files
--from-doc <file>Bootstrap from an existing document (PRD, spec, etc.)
--clean-rootMove auxiliary docs to .claude/docs/

Git Safety

FlagDescription
--no-git-checkSkip git safety entirely
--git-mode ask|stash|skipGit safety mode (default: ask)

Template System

FlagDescription
--template-dir <dir>Template overrides directory (default: .claude-primer/templates/)

Watch Mode

FlagDescription
--watchWatch for changes and suggest updates (poll-based)
--watch-interval <secs>Polling interval in seconds (default: 5)
--watch-autoAuto-regenerate without confirmation

Multi-Agent Output

FlagDescription
--agent <agents>Target agents: claude, cursor, copilot, windsurf, aider, codex, all (comma-separated)
--format markdown|yaml|jsonOutput format for agent context files (default: markdown)

Plugin System

FlagDescription
--plugin-dir <dir>Plugin generators directory (default: .claude-primer/plugins/)

Telemetry

Flag / Env VarDescription
--telemetry-offDisable telemetry even if env var is set
CLAUDE_PRIMER_TELEMETRY=1Opt-in to anonymous usage analytics

Examples

# Full automation: no prompts, overwrite all, skip git
claude-primer /path/to/repo --force-all --yes --no-git-check

# Generate for all AI tools
claude-primer --agent all --yes

# Watch with auto-regeneration and custom templates
claude-primer --watch --watch-auto --template-dir ./templates

# CI/CD pipeline usage
claude-primer /workspace --yes --no-git-check --plan-json > analysis.json
