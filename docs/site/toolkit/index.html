One CLI. Full setup.

Claude Toolkit is the unified meta-package that combines Claude Primer and MAO Orchestrator. Prime your codebase and install multi-agent workflows with a single command.

Primer — Knowledge MAO — Orchestration Toolkit — Unified

Commands

claude-toolkit init
Prime project + install MAO agents in one step
claude-toolkit doctor
Health check for all components (Primer docs, MAO agents, config)
claude-toolkit update
Update all tools to latest version
claude-toolkit primer [args]
Passthrough to claude-primer
claude-toolkit mao [args]
Passthrough to mao-orchestrator

How It Works

Running claude-toolkit init triggers a two-phase setup that fully prepares your project for AI-assisted development.

1

Primer Phase

Scans your codebase and generates context-aware documentation — CLAUDE.md, STANDARDS.md, QUICKSTART.md, and ERRORS_AND_LESSONS.md. These files give Claude deep understanding of your project.

claude-primer
2

MAO Phase

Installs the multi-agent orchestrator: slash commands (/mao, /mao-plan, /mao-status) and agent definitions (Architect, Orchestrator, Implementer, Worker, Verifier, Reviewer, Reflector, Explorer).

mao-orchestrator

Install

Terminal
# Quick start (no install needed)
npx claude-toolkit init

# Or install globally
npm install -g claude-supertools

# Then run from any project
claude-toolkit init
