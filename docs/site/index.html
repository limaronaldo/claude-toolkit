Claude Toolkit

A suite of tools for AI-assisted development with Claude Code. Prime your codebase, orchestrate multi-agent workflows, or use both together.

Claude Primer

Scans your codebase and generates context-aware knowledge architecture — CLAUDE.md, STANDARDS.md, QUICKSTART.md, and ERRORS_AND_LESSONS.md.

Knowledge

MAO Orchestrator

Multi-Agent Orchestrator with intelligent Opus/Sonnet/Haiku model tiering, DAG-based task scheduling, and git worktree parallelism.

Orchestration

Claude Supertools

Unified CLI that combines Primer + MAO in one command. Prime your project and install agents with a single claude-toolkit init.

Unified

Quick Start

# Full setup (recommended)
npx claude-toolkit init

# Or install individually
npx claude-primer              # Generate knowledge docs
npx mao-orchestrator init      # Install multi-agent orchestrator
