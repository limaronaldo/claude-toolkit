VS Code Extension

The Claude Primer VS Code extension brings knowledge architecture generation directly into your editor.

Installation

# From VS Code Marketplace (coming soon)
code --install-extension limaronaldo.claude-primer

# Or from source
cd vscode/ && vsce package && code --install-extension claude-primer-*.vsix

Requires the claude-primer CLI to be installed and available on your PATH (or via npx/pipx).

Commands

All commands are available via the Command Palette (Cmd+Shift+P / Ctrl+Shift+P):

Command Description
Claude Primer: Generate Run full generation with --yes --git-mode skip
Claude Primer: Dry Run Preview what would be generated without writing files
Claude Primer: Show Diff Show unified diff of changes, opens as a .diff file
Claude Primer: Init Config Create .claude-primer.toml via guided prompts (no terminal needed)
Claude Primer: Plan JSON View project analysis in a webview panel

Binary Resolution

The extension finds claude-primer automatically:

  1. Checks PATH for claude-primer
  2. Falls back to npx --yes claude-primer
  3. Falls back to pipx run claude-primer

Output

  • Generate and Dry Run stream output to the "Claude Primer" output channel
  • Show Diff opens a syntax-highlighted .diff file
  • Plan JSON renders a formatted webview with project metadata
